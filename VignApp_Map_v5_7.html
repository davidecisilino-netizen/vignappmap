Invia dati al PC

Genera un file .json con tutti i tuoi rilievi. Importalo nel software PC tramite la scheda Sincronizza.

Ricevi dati dal PC

Importa il file generato dal software PC (scheda Sincronizza → Esporta per Smartphone). I dati vengono uniti senza sovrascrivere quelli esistenti.

Come sincronizzare
  1. Dal PC: crea vigneti → "Esporta per Smartphone"
  2. Trasferisci il .json su iPhone (email, AirDrop)
  3. Qui su smartphone: "Carica JSON dal PC" → i vigneti appaiono
  4. Esegui i rilievi in campo con l'app
  5. Qui: "Esporta backup completo" → condividi al PC
  6. Dal PC: Sincronizza → "Importa da Smartphone"
